(2) Certified Machine Learning Workshop - Google MLCC syllabus.

This workshop will use Google's Online Code Laboratories to teach Machine Learning. This is an entry-level workshop to deepen the fundamental understanding of ML.

Batch Size of 20 candidates.

Data Set
  • New York Housing Data
  • Health care disease data set
  • Financial risk/product data
Trainer:
Siddhant Wade
Siddhant is the founder of Big Data Mumbai, Mumbai's most organic community of Data Science professionals. He's an excellent analyst and well connected with the local data science trends and influencers.
He frequently hosts training sessions for students and professionals on data science subjects.

The workshop will require candidates to read some preparation material.

Modules:
Day 1:
  • Framing
  • Linear Regression and Training and Loss
  • Reducing LossTensorflow basics and toolkit
  • Generalization Techniques
  • Validation Set
  • Representation
  • Feature Crossing and Optimisation
  • Regularisation - L2 and Lambda
Day2:
  • Logistic Regression - Probability Calculation
  • Classification - Precision, Accuracy, Recall, ROC and AUC, Prediction Bias study
  • Regularisation sparsity
  • Neural networks
  • Training neural nets
  • Multiclass neural nets - softmax
  • Embeddings
  • ML engineering fundamentals - production level deployment and stack
